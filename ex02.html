HTML : Hyper Text Markup Language
- 다른 페이지에 링크가 걸린 문서를 작성하는 언어
- HTML은 <>로 이루어진 문법을 사용한다
- <>를 Tag라고 하고 Tag에 둘러 쌓이면 다양한 효과를 낸다
- Tag에 둘러 쌓인 부분을 요소(Elemets)라고 한다
- Tag는 여는(Open) 태그와 닫는(Close) 태그로 이루어진다
- 닫는 태그는 / 가 앞에 붙고, 닫는 태그가 없는 태그들도 있다
- 이러한 태그는 웹 브라우저가 해석해서 화면에 출력하게 된다
web : HTTP 프로토콜로 제공되는 서비스를 의미
- 서버에서 작성한 웹 페이지를 클라이언트에게 제공하는 서비스
- WWW는 World Wide Web의 약자로 전 세계적으로 제공되는 웹을 의미
